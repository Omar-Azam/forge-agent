CLI Reference

Forge Agent provides over 40 command-line flags for fine-grained control over the agent's behavior.

Core Options

--interactive, -i

Start interactive REPL mode where you can run multiple tasks in a shared AI context.

forge-agent --interactive

--dir <path>, -d

Set the working directory for the agent (default: current directory).

forge-agent --dir ~/projects/my-app "add a readme"

--model <name>, -m

Specify the AI model to use. Options: deepseek, chatgpt, gemini.

forge-agent --model=chatgpt "write a blog post"

--profile <name>

Activate a pre-configured agent profile.

forge-agent --profile=backend "add auth"

--plan

Show a step-by-step execution plan before the agent starts acting.

--think

Display DeepSeek R1's chain-of-thought reasoning process.

Memory & Sessions

--resume

Resume a previous task via an interactive selection list.

--resume=last

Immediately resume the most recently run task.

--rerun

Re-run the most recent task from scratch.

--history

Browse and resume past tasks from a list.

Diagnostics & Security

--benchmark

Run the performance benchmark suite.

--diagnostics

Generate a system report for bug debugging.

--security

Show the current security posture and blocked paths.

Support & Launch

--sponsor

Show sponsorship options and tiers.

--no-sponsor-nudge

Disable the funding reminder for the current run.

--launch-assets

Generate launch materials for social media.

--examples

Open the interactive example project gallery.

Templates

--template=<name>

Run a saved task template.

Output & Format

--format=<name>

Set output format: text, markdown, json, minimal, silent.

Watch Mode

--watch

Watch files for changes and re-run the task automatically.

Performance

--max-iterations=<N>

Maximum steps before stopping (default: 100).

Configuration

--setup

Run the interactive setup wizard.
