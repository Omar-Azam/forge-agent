Getting Started

Prerequisites

  • Node.js 18 or higher
  • A free account on DeepSeek, ChatGPT, or Gemini
  • npm (comes with Node.js)

Installation

Install Forge Agent globally via npm:

npm install -g @omar-azam/forge-agent

Verify the installation:

forge-agent --version   # should print: Forge Agent v1.4.0

First Time Setup

Run the interactive setup wizard to configure your preferred AI model and other settings:

forge-agent --setup

The wizard will guide you through:

  • Choosing a default model (DeepSeek, ChatGPT, or Gemini)
  • Setting your working directory
  • Configuring headless mode
  • Enabling/disabling persistent memory

Your First Task

Run a single task immediately from the CLI:

forge-agent "create a hello world HTML page"

Forge Agent will open the browser, send the task to the AI, and begin executing the necessary tool calls to create the file.

Interactive Mode

For complex projects, use interactive mode to run multiple tasks with a shared AI context:

forge-agent --interactive
  • Type your tasks one by one.
  • Type new to start a fresh chat session.
  • Type exit or press Ctrl+C to quit.

Understanding the Output

Forge Agent uses a structured Terminal UI (TUI) to show progress:

  • ⚡ TOOL CALL: The agent is calling a tool (e.g., write_file).
  • ✓ Result: The tool executed successfully.
  • ✗ Result: The tool failed (the agent will usually try to fix it).
  • Step N/100: Current iteration count.
💡 Tip: For slow internet or slow machines, you can increase the response timeout in your config: 
{ "RESPONSE_TIMEOUT": 900000, "MAX_ITERATIONS": 150 }
