Installation

npm install -g @omar-azam/forge-agent@2.0.0

# First time setup
forge-agent --setup

# Run your first task
forge-agent "build a REST API with Express"

How it works

1. 📝 You give Forge Agent a task in plain English

2. 🌐 It opens the AI website in a browser automatically

3. 🔧 Receives tool calls (write file, run command, etc.)

4. ⚡ Executes them on your machine

5. 🔄 Feeds results back and repeats until done
