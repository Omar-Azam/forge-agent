Forge Agent Roadmap: What's Coming in 2026

With the release of Forge Agent v1.4.0, we've achieved a major milestone in autonomous AI coding. But this is only the beginning. Today, we're sharing our roadmap for the rest of 2026 as we move towards v2.0.0.

Where We Are Today (v1.4.0)

Across 48 days of development, we've built a foundation that is unique in the AI coding space. We have 37+ built-in tools, 1150+ passing tests, and a robust browser-based architecture that bypasses the need for expensive API keys. Forge Agent is already capable of building entire REST APIs, React components, and CLI tools autonomously.

v1.5.0 — Scale & Integration (Q2 2026)

Our next phase focuses on making Forge Agent more powerful and easier to integrate into existing workflows.

  • GitHub Container Registry: Official, pre-built Docker images will be available on GHCR, making it a one-command setup for any environment.
  • Expanded Toolbox: New tools for handling PDFs, large CSV datasets, and making direct API calls will broaden the scope of tasks the agent can handle.
  • Streaming Response Display: You'll be able to see the AI's "thoughts" and tokens in real-time as they are generated, rather than waiting for the final response.
  • Parallel Execution: The agent will be able to run independent tool calls simultaneously, significantly speeding up complex tasks.

v2.0.0 — Platform (Q3-Q4 2026)

In the second half of the year, Forge Agent will evolve from a CLI tool into a comprehensive platform for autonomous development.

  • Multi-Agent Coordination: Run multiple agents in parallel on the same task. Imagine a "Project Manager" agent delegating work to "Frontend" and "Backend" sub-agents.
  • GUI Desktop Application: A full-featured desktop app built with Electron or Tauri, featuring a live browser view so you can watch the agent work in real-time.
  • API Server Mode: Run Forge Agent as a service with a REST and WebSocket API, allowing you to trigger agent tasks from webhooks or other external events.
  • Local Model Support: Integration with Ollama and other local model providers, allowing for 100% offline, private autonomous coding.

How to Influence the Roadmap

Forge Agent is a community-driven project. You can help shape our direction by:

  • Voting: Participate in polls and discussions on our GitHub Discussions page.
  • Requesting: Open a feature request issue on GitHub if you have an idea for a new tool or capability.
  • Sponsoring: Financial support allows the maintainers to prioritize specific roadmap items and dedicate more time to development.

How to Stay Updated

The best way to stay in the loop is to Watch our GitHub repository and check this blog regularly. We publish detailed release notes for every version, and our roadmap is always live in the root of our repo.

Let's build the future of software engineering, one autonomous task at a time.
