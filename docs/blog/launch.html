Introducing Forge Agent: Autonomous AI Coding with No API Key

After 48 days of intensive development and over 1150 automated tests, I am thrilled to announce the public launch of Forge Agent — the first truly free, autonomous AI coding agent that drives web-based AI assistants without requiring a single API key.

The Problem: The AI Tax

AI coding tools are transforming how we build software, but they come with a significant barrier: cost. GitHub Copilot, Cursor, and Devin all require monthly subscriptions or expensive pay-per-task API keys. For students, hobbyists, and developers in emerging markets, state-of-the-art AI assistance is often economically out of reach.

The Idea: Driving the UI

DeepSeek, ChatGPT, and Gemini all provide world-class AI for free via their web interfaces. They are brilliant, but they lack free API access for automation. However, a web browser doesn't care who is clicking the buttons. Forge Agent automates the browser to give you the power of autonomous coding for $0.

What Forge Agent Does

Forge Agent is a Node.js CLI tool that takes a task in plain English and executes it autonomously. It reads your files, writes new code, runs your test suite, and debugs errors until the job is done.

forge-agent "build a REST API with Express and JWT auth"

It doesn't just suggest code; it implements it. It handles the "boring" parts of development so you can focus on architecture and logic.

48 Days of Hardening

This isn't a prototype. We've spent 48 days building a robust production-ready tool:

  • 37+ Built-in Tools: From file I/O and Git to process management and semantic search.
  • 1150+ Automated Tests: Ensuring every tool and parser strategy works across platforms.
  • Multi-Model Support: Swap between DeepSeek, ChatGPT, and Gemini with a single flag.
  • Enterprise-Ready: Full Docker support, security sandbox, and audit logging.

How It Works (Technical)

Forge Agent uses Playwright to drive a headless Chromium instance. It maintains a persistent logged-in session, meaning you log in once and the agent can run indefinitely. AI responses are parsed using a proprietary 6-strategy engine that extracts tool calls from markdown, XML, or even bare JSON. These tools then execute directly on your local filesystem, and the results are fed back into the AI to complete the loop.

Getting Started

You can install Forge Agent globally via npm:

npm install -g @omar-azam/forge-agent
forge-agent --setup
forge-agent "your first task"
