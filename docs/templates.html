Task Templates

Templates are saved, reusable task descriptions for common development workflows. Instead of typing a long prompt every time, you can trigger a template with a single flag.

Built-in Templates

Forge Agent includes 10 built-in templates for common setup and maintenance tasks:

add-typescript

Add TypeScript to an existing JavaScript project. Installs dependencies and configures tsconfig.json.

forge-agent --template=add-typescript

add-jest

Set up Jest testing framework from scratch. Creates configuration and example tests.

forge-agent --template=add-jest

add-docker

Create a production-ready Dockerfile and docker-compose.yml for the project.

forge-agent --template=add-docker

fix-tests

Run the test suite and autonomously fix any failing tests by editing the source code.

forge-agent --template=fix-tests

code-review

Review the codebase for security issues, performance bottlenecks, and code quality.

forge-agent --template=code-review

Using Templates

Run any template by name using the --template flag:

forge-agent --template=add-eslint

List all available templates (built-in and custom):

forge-agent --list-templates

Show full details of a specific template:

forge-agent --show-template=add-jest

Creating Custom Templates

You can save your own tasks as custom templates for future use:

forge-agent --save-template=my-setup "Add our standard boilerplate..."

Custom templates are stored in ~/.deepseek-agent/templates.json.

Template Variables

Templates can use {{variable}} syntax for dynamic substitution. Provide values using the --vars flag as a JSON string:

# Template: "Create a component named {{name}}"
forge-agent --template=create-comp --vars='{"name": "MyHeader"}'
