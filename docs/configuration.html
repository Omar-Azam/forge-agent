Configuration

Config File Locations

Forge Agent loads configuration in the following order of priority (highest wins):

  1. Project Config: deepseek-agent.config.json in the working directory.
  2. Global Config: ~/.deepseek-agent/config.json in your home directory.
  3. Defaults: Internal hardcoded defaults.

All Config Keys

KeyTypeDefaultDescription
MODELstring"deepseek"AI model: deepseek, chatgpt, gemini.
MAX_ITERATIONSnumber100Max steps before the agent stops.
RESPONSE_TIMEOUTnumber600000Timeout for AI responses (ms).
HEADLESSbooleanfalseRun browser in invisible mode.
MEMORY_ENABLEDbooleantruePersist project memory across runs.
STRICT_SANDBOXbooleanfalseRestrict file access to working dir.
ALLOW_DANGEROUS_COMMANDSbooleanfalseAllow rm -rf and similar commands.
ALLOW_ENV_PLAIN_READbooleanfalseSend .env files unmasked to AI.
AUDIT_LOGbooleanfalseLog all tool calls to audit.log.
PARAM_VALIDATIONbooleantrueValidate tool parameters before execution.
PLANNING_MODEbooleanfalseProduce execution plans.
NO_TUIbooleanfalseDisable the terminal UI.
DISABLE_SPONSOR_NUDGEbooleanfalseHide the small donation reminder.
DOCKER_CONTAINERstring""Internal ID when running inside Docker.
OUTPUT_FORMATstring"text"Result format: text, json, markdown.

Setup Wizard

The easiest way to configure Forge Agent is through the setup wizard:

forge-agent --setup

Example Configs

Minimal / CI Mode

{
  "HEADLESS": true,
  "NO_TUI": true,
  "MEMORY_ENABLED": false
}

Slow Machine / Connection

{
  "RESPONSE_TIMEOUT": 1200000,
  "MAX_ITERATIONS": 150,
  "STABLE_DELAY": 3000
}
